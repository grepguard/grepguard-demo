# GrepGuard

**AI-powered security reviews for AI-generated code.**

GrepGuard analyzes code written by AI assistants (Cursor, Claude Code, GitHub Copilot) and identifies security vulnerabilities, common anti-patterns, and policy violations before they reach production.
